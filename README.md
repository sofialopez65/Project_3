# Project_3
This project analyzes transportation trends in the United States using the National Household Travel Survey (NHTS) and Next Generation Simulation (NGSIM) datasets. The goal is to generate clear visualizations—including bar charts, histograms, boxplots, and time-series plots—to highlight key patterns in travel behavior and driving dynamics. Additionally, an Intelligent Driver Model (IDM) simulation is implemented to model and visualize vehicle trajectories. The repository includes all Python code, datasets, and documentation needed to reproduce the analysis, along with a user guide for running the scripts. Findings from this project provide insight into transportation system performance and support data-driven decision-making in transportation planning.
